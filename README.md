# Since PWA

Starter React + TypeScript + Vite avec:

- Tailwind CSS
- Composants style shadcn/ui
- Supabase client
- Configuration PWA (vite-plugin-pwa)

## Démarrage

```bash
npm install
cp .env.example .env
npm run dev
```

## Supabase

Mettez à jour `VITE_SUPABASE_URL` dans `.env` avec l'URL de votre projet.
